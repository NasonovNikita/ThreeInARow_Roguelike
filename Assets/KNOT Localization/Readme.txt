### GETTING STARTED

Open Tools > KNOT Localization > About > Online or Offline Manual to get started.

Mark KnotDatabase_demo as Default in "Project Settings/KNOT Localization" and open "KNOT Localization/Addons/Demo/KnotLocalization_demo" to run the demo scene.
